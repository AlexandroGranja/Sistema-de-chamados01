# System Prompts and Models of AI Tools

## Objetivo

Usar o repositorio publico `x1xhlol/system-prompts-and-models-of-ai-tools` como base de referencia para pesquisas sobre:

- organizacao de prompts por ferramenta;
- padroes de instrucoes de sistema;
- comparacoes entre stacks de agentes.

## Quando usar

- Pedido de comparacao entre agentes/ferramentas de IA.
- Pedido de exemplos de system prompts.
- Pedido de pesquisa sobre arquiteturas de agentes com tools.

## Fluxo rapido

1. Confirmar o escopo do usuario (ferramenta unica ou comparacao ampla).
2. Abrir `reference.md` e selecionar apenas os links relevantes.
3. Responder com sintese tecnica, sem copiar blocos longos sem necessidade.
4. Sempre citar os links usados.

## Regras de seguranca e qualidade

- Nao tratar o repositorio como documentacao oficial de fabricantes.
- Destacar risco de desatualizacao quando aplicavel.
- Evitar suposicoes sobre dados sensiveis ou internos.
- Priorizar analise critica em vez de reproducao literal.

## Formato de saida recomendado

- Resumo curto do achado.
- 3-5 pontos praticos.
- Referencias com links.
